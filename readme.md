# terminal-link-cli

> Create clickable links in the terminal

<img src="screenshot.gif" width="301">

## Install

```
$ npm install --global terminal-link-cli
```

## Usage

```
$ terminal-link --help

  Usage
    $ terminal-link <text> <url>

  Example
    $ terminal-link 'My Website' 'https://sindresorhus.com'
```
